# Заготівля лісу

בבוקר קריר צוות קטן של עובדי יער נכנס לחלקת יער שבה עומדים לכרות כמה עצים בוגרים.
לפני תחילת העבודה המנהל מסביר איך בודקים את מצב הקרקע, את כיוון הרוח ואת המרחק בין העצים.
הוא גם מזכיר שחשוב להכיר את חלקי העץ: השורשים נמצאים באדמה, הגזע נושא את המשקל, והענפים והעלים גדלים למעלה בצמרת.
כאשר צריך להכין את העץ לכריתה, העובדים קודם גוזמים ענפים נמוכים ומנקים את האזור מסביב לגזע.
אחר כך הם מודדים את הקוטר של הגזע ומסמנים את כיוון הנפילה הבטוח.
עובד אחד מפעיל מסור שרשרת, ועובד אחר עומד ליד מכונת יער כבדה שמסוגלת להרים את הגזעים ולגרור אותם לשביל.
אחרי שהעץ נופל, חותכים את הגזע לקטעים ארוכים, מסירים ענפים דקים ומעמיסים את בולי העץ על משאית.
המשאית מובילה את המטען למנסרה, ושם מתחיל שלב אחר של העבודה.
במנסרה חותכים את בולי העץ ללוחות, לקורות ולקרשים, לפי העובי, האורך והאיכות של החומר.
חלק מהעץ מתאים לבניית גגות ורצפות, וחלק אחר טוב לרהיטים, למדפים או למסגרות פשוטות.
בסוף היום העובדים מנקים את הכלים, בודקים שוב את הציוד וכותבים דוח קצר על מספר העצים, סוגי העץ והכמות שהועברה למנסרה.

- כריתת יער / כריתת עצים - заготівля лісу / вирубка дерев

- עץ בוגר - доросле дерево
- שורשים - коріння
- גזע - стовбур
- ענף / ענפים - гілка / гілки
- צמרת - крона дерева
- לגזום - обрізати гілки
- קוטר - діаметр
- כיוון הנפילה - напрямок падіння
- מסור שרשרת - бензопила
- מכונת יער - лісозаготівельна машина
- לגרור - тягнути, волочити
- בול עץ / בולי עץ - колода / колоди
- מנסרה - тартак, лісопилка
- לוח - дошка, плита
- קורה - балка
