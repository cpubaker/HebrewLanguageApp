# Іврит у класі

בְּשִׁעוּר עִבְרִית הַמּוֹרָה עוֹמֶדֶת לְיַד הַלּוּחַ.
הִיא אוֹמֶרֶת, "פִּתְחוּ אֶת הַסֵּפֶר בְּבַקָּשָׁה."
הַתַּלְמִידִים פּוֹתְחִים אֶת הַסְּפָרִים וְקוֹרְאִים יַחַד.
אַחַר כָּךְ הַמּוֹרָה כּוֹתֶבֶת מִלָּה חֲדָשָׁה עַל הַלּוּחַ.

הִיא שׁוֹאֶלֶת, "מַה זֶּה?"
תַּלְמִיד אֶחָד עוֹנֶה, וְהַמּוֹרָה מְחַיֶּכֶת.
כֻּלָּם מַקְשִׁיבִים וְלוֹמְדִים מִלִּים חֲדָשׁוֹת.

## Основні слова

- עִבְרִית - іврит
- מוֹרָה - вчителька
- לוּחַ - дошка
- סֵפֶר / סְּפָרִים - книга / книги
- מִלָּה - слово
- חָדָשׁ / חֲדָשָׁה - новий / нова
- תַּלְמִיד - учень
- בְּבַקָּשָׁה - будь ласка
- יַחַד - разом
