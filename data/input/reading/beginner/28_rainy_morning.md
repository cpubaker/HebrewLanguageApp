# Дощовий ранок

בַּבֹּקֶר יֵשׁ גֶּשֶׁם חָזָק.
דָּוִד מַבִּיט בַּחַלּוֹן וְרוֹאֶה עָנָנִים אֲפֹרִים.
הוּא לוֹבֵשׁ מְעִיל וְנוֹעֵל מַגָּפַיִם.
הוּא לוֹקֵחַ מִטְרִיָּה גְּדוֹלָה, כִּי הוּא צָרִיךְ לָצֵאת.

בַּחוּץ יֵשׁ שְׁלוּלִית לְיַד הַדֶּלֶת.
דָּוִד הוֹלֵךְ לְאַט וְלֹא נִכְנָס לַשְּׁלוּלִית.
הוּא מַגִּיעַ בַּזְּמַן וְשָׂמֵחַ שֶׁיֵּשׁ לוֹ מִטְרִיָּה.

## Основні слова

- גֶּשֶׁם - дощ
- חָזָק - сильний
- חַלּוֹן - вікно
- עָנָן / עָנָנִים - хмара / хмари
- אָפֹר / אֲפֹרִים - сірий / сірі
- מְעִיל - пальто, куртка
- מַגָּפַיִם - чоботи
- מִטְרִיָּה - парасоля
- שְׁלוּלִית - калюжа
- מַיִם - вода
- בַּזְּמַן - вчасно
