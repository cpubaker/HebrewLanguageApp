# Відпочинок

‏אֲנִי יוֹצֵא‏
‏לְחֻפְשָׁה בַּקַּיִץ.‏
‏אֲנִי נוֹסֵעַ‏
‏לְמָקוֹם חַם וְיָפֶה.‏
‏יֵשׁ שָׁם‏
‏חוֹף, שֶׁמֶשׁ וְיָם.‏
‏אֲנִי אוֹהֵב‏
‏לִשְׂחוֹת וְלְהִתְרַחֵץ.‏
‏אֲנִי יָשֵׁן‏
‏בְּמָלוֹן קָטָן.‏
‏בַּתְּחִלָּה אֲנִי‏
‏מַגִּיעַ לְשְׂדֵה הַתְּעוּפָה.‏
‏אַחַר כָּךְ אֲנִי‏
‏עוֹלֶה לַמָּטוֹס‏
‏עִם מִזְוָדָה.‏

## Основні слова

- חֻפְשָׁה - відпочинок, відпустка
- קַיִץ - літо
- נוֹסֵעַ - їде, подорожує
- מָקוֹם - місце
- חַם - теплий, жаркий
- יָפֶה - гарний
- חוֹף - пляж
- שֶׁמֶשׁ - сонце
- יָם - море
- לִשְׂחוֹת - плавати
- לְהִתְרַחֵץ - купатися
- מָלוֹן - готель
- קָטָן - маленький
- תְּחִלָּה - початок, спочатку
- מַגִּיעַ - прибуває
- שְׂדֵה הַתְּעוּפָה - аеропорт
- אַחַר כָּךְ - потім
- עוֹלֶה - сідає, піднімається
- מָטוֹס - літак
- מִזְוָדָה - валіза
