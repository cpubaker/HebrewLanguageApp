# Створювати

## Інфінітив

- לִיצוֹר (ліцóр)

## Часті форми

- теперішній час: יוֹצֵר / יוֹצֶרֶת (йоцéр / йоцéрет)
- минулий час: יָצַרְתִּי, יָצַר, יָצְרָה (яцáрті, яцáр, яцрá)
- майбутній час: אֶצּוֹר, יִצּוֹר, נִצּוֹר (ецóр, їцóр, ніцóр)

## Короткі приклади

- אֲנִי יוֹצֵר רְשִׁימָה חֲדָשָׁה. — Я створюю новий список.
- הִיא יָצְרָה אֲוִירָה נְעִימָה. — Вона створила приємну атмосферу.
- מָחָר נִצּוֹר מַשֶּׁהוּ חָדָשׁ. — Завтра ми створимо щось нове.
