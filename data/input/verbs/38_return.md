# Повертатися

## Інфінітив

- לַחֲזוֹר (лахазóр)

## Часті форми

- теперішній час: חוֹזֵר / חוֹזֶרֶת (хозéр / хозéрет)
- минулий час: חָזַרְתִּי, חָזַר, חָזְרָה (хазáрті, хазáр, хазрá)
- майбутній час: אֶחֱזוֹר, יַחֲזוֹר, נַחֲזוֹר (ехезóр, яхазóр, нахазóр)

## Короткі приклади

- אֲנִי חוֹזֵר הַבַּיְתָה. — Я повертаюся додому.
- הִיא חָזְרָה מֵהָעֲבוֹדָה. — Вона повернулася з роботи.
- מָחָר נַחֲזוֹר לְכָאן. — Завтра ми повернемося сюди.
