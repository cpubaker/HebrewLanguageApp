# Guide Content Instructions

## Purpose
- This folder contains numbered guide lessons for the local Hebrew learning app.
- Files here should stay easy for the Tkinter guide viewer to load and render.

## What Counts As A Lesson
- Lesson files should use numbered names such as `01_intro_alphabet.md` or `17_numbers_time_dates.md`.
- Keep existing lesson numbers and filenames stable unless a rename is explicitly required.
- Use UTF-8 encoding.

## Required File Shape
- Start the file with a Markdown heading like `# Lesson Title`.
- The first heading is used as the displayed lesson title.
- Put the lesson content after the title.
- Empty files are treated as missing content and should be avoided.

## Writing Style
- Keep explanations in clear Ukrainian for beginner-friendly learning flow.
- When useful, introduce Hebrew examples together with transliteration and Ukrainian meaning.
- Prefer short sections with `##` and `###` headings, short paragraphs, and simple bullet lists.
- Prefer progressive explanation: rule first, then a few examples, then a small pattern or reminder.

## Formatting Rules
- Use only simple Markdown that the app already renders well: headings, paragraphs, and lists.
- Avoid HTML, tables, and complex Markdown features.
- Keep spacing clean and readable.

## Content Safety
- Stay aligned with neighboring lessons in tone and difficulty.
- Do not turn one lesson into several unrelated topics.
- If you add examples, keep Hebrew spelling, transliteration, and Ukrainian glosses consistent inside the file.

## Support Files
- This folder may also contain helper files like this `AGENTS.md`.
- Helper files are for contributors and should not be treated as lessons.
