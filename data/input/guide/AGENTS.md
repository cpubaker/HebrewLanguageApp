# Guide Content Instructions

## Scope
- This folder contains active numbered guide articles only.
- Keep durable guide content here; do not keep drafts, archives, or temporary split files in this directory.

## File Rules
- Use stable numbered filenames such as `23_basic_prepositions.md`.
- Do not renumber or rename guide files unless the task explicitly requires it.
- Use UTF-8.
- Start each file with a Markdown heading. The first heading is the displayed title.
- Avoid empty files.

## Writing Style
- Write clear, natural Ukrainian for a learner using the app as a reference.
- Treat the guide as a lookup module, not a lesson script.
- Prefer short sections, short paragraphs, and simple bullet lists.
- Lead with the rule, contrast, or model first; then show short examples.
- Keep Hebrew, transliteration, and Ukrainian glosses consistent inside one file.
- Explain grammar terms in everyday language if you use them.

## Structure
- Keep one central topic per file.
- If the topic is broad, split it into tight subsections rather than mixing several themes.
- A typical article should include:
  - short intro (`Коротко`)
  - main model or contrast
  - forms / variants if needed
  - short examples
  - common mistakes where useful
  - related topics

## Do Not Add
- Full lesson flow
- drills or homework blocks
- spaced repetition advice
- mini-dialogue sections unless explicitly requested
- HTML, tables, or complex Markdown

## Editorial Priorities
- Favor correctness over flourish.
- Favor natural phrasing over textbook prose.
- Favor short, retrievable explanations over long paraphrases.
- Call out common Ukrainian-to-Hebrew interference when it prevents mistakes.

## Notes
- Helper files like this `AGENTS.md` are not lessons.
- Stay aligned with neighboring guide files in tone and difficulty.
