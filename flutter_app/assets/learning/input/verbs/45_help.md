# Допомагати

## Інфінітив

- לַעֲזוֹר (лаазóр)

## Часті форми

- теперішній час: עוֹזֵר / עוֹזֶרֶת (озéр / озéрет)
- минулий час: עָזַרְתִּי, עָזַר, עָזְרָה (азáрті, азáр, азрá)
- майбутній час: אֶעֱזוֹר, יַעֲזוֹר, נַעֲזוֹר (еезóр, яазóр, наазóр)

## Короткі приклади

- הוּא עוֹזֵר לִי הַרְבֵּה. — Він дуже мені допомагає.
- אֲנִי עָזַרְתִּי לָהּ אֶתְמוֹל. — Я допоміг їй учора.
- מָחָר נַעֲזוֹר לָכֶם. — Завтра ми допоможемо вам.
