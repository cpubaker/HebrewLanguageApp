# Користуватися

## Інфінітив

- לְהִשְׁתַּמֵּשׁ (легіштамéш)

## Часті форми

- теперішній час: מִשְׁתַּמֵּשׁ / מִשְׁתַּמֶּשֶׁת (міштамéш / міштамéшет)
- минулий час: הִשְׁתַּמַּשְׁתִּי, הִשְׁתַּמֵּשׁ, הִשְׁתַּמְּשָׁה (гіштамáшті, гіштамéш, гіштамшá)
- майбутній час: אֶשְׁתַּמֵּשׁ, יִשְׁתַּמֵּשׁ, נִשְׁתַּמֵּשׁ (ештамéш, їштамéш, ніштамéш)

## Короткі приклади

- אֲנִי מִשְׁתַּמֵּשׁ בַּמִּלּוֹן הַזֶּה. — Я користуюся цим словником.
- הִיא הִשְׁתַּמְּשָׁה בַּמַּפָּה. — Вона користувалася картою.
- מָחָר נִשְׁתַּמֵּשׁ בְּדֻגְמָאוֹת חֲדָשׁוֹת. — Завтра ми використаємо нові приклади.
