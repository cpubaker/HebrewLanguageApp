# Починати

## Інфінітив

- לְהַתְחִיל (легатхíл)

## Часті форми

- теперішній час: מַתְחִיל / מַתְחִילָה (матхíл / матхілá)
- минулий час: הִתְחַלְתִּי, הִתְחִיל, הִתְחִילָה (гітхáлті, гітхíл, гітхілá)
- майбутній час: אַתְחִיל, יַתְחִיל, נַתְחִיל (атхíл, ятхíл, натхíл)

## Короткі приклади

- הַשִּׁיעוּר מַתְחִיל עַכְשָׁיו. — Урок починається зараз.
- אֲנִי הִתְחַלְתִּי לִלְמֹד עִבְרִית. — Я почав вивчати іврит.
- מָחָר נַתְחִיל פֶּרֶק חָדָשׁ. — Завтра ми почнемо новий розділ.
