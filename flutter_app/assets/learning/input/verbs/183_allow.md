# Дозволяти

## Інфінітив

- לְהַרְשׁוֹת (легаршóт)

## Часті форми

- теперішній час: מַרְשֶׁה / מַרְשָׁה (маршé / маршá)
- минулий час: הִרְשֵׁיתִי, הִרְשָׁה, הִרְשְׁתָה (гіршéйті, гіршá, гірштá)
- майбутній час: אַרְשֶׁה, יַרְשֶׁה, נַרְשֶׁה (аршé, яршé, наршé)

## Короткі приклади

- הִיא מַרְשָׁה לַיְלָדִים לְשַׂחֵק. — Вона дозволяє дітям гратися.
- הוּא הִרְשָׁה לִי לְהִכָּנֵס. — Він дозволив мені зайти.
- מָחָר נַרְשֶׁה לְכֻלָּם לִשְׁאֹל. — Завтра ми дозволимо всім ставити запитання.
