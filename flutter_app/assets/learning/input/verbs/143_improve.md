# Покращувати

## Інфінітив

- לְשַׁפֵּר (лешапéр)

## Часті форми

- теперішній час: מְשַׁפֵּר / מְשַׁפֶּרֶת (мешапéр / мешапéрет)
- минулий час: שִׁפַּרְתִּי, שִׁפֵּר, שִׁפְּרָה (шипáрті, шипéр, шипрá)
- майбутній час: אֲשַׁפֵּר, יְשַׁפֵּר, נְשַׁפֵּר (ешапéр, єшапéр, нешапéр)

## Короткі приклади

- אֲנִי מְשַׁפֵּר אֶת הָעִבְרִית שֶׁלִּי. — Я покращую свою іврит.
- הוּא שִׁפֵּר אֶת הַתּוֹצָאָה. — Він покращив результат.
- מָחָר נְשַׁפֵּר אֶת הַמַּעֲרֶכֶת. — Завтра ми покращимо систему.
