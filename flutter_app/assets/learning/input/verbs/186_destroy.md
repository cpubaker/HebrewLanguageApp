# Руйнувати

## Інфінітив

- לַהֲרוֹס (лагарóс)

## Часті форми

- теперішній час: הוֹרֵס / הוֹרֶסֶת (горéс / горéсет)
- минулий час: הָרַסְתִּי, הָרַס, הָרְסָה (гара́сті, гара́с, гарсá)
- майбутній час: אֶהֱרוֹס, יַהֲרוֹס, נַהֲרוֹס (егерóс, ягарóс, нагарóс)

## Короткі приклади

- הָרוּחַ הוֹרֶסֶת אֶת הַסֻּכָּה. — Вітер руйнує сукку.
- הוּא הָרַס אֶת הַקִּיר הַיָּשָׁן. — Він зруйнував стару стіну.
- מָחָר לֹא נַהֲרוֹס אֶת מַה שֶּׁבָּנִינוּ. — Завтра ми не зруйнуємо те, що збудували.
