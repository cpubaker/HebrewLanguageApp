# Плакати

## Інфінітив

- לִבְכּוֹת (лівкóт)

## Часті форми

- теперішній час: בּוֹכֶה / בּוֹכָה (бохé / бохá)
- минулий час: בָּכִיתִי, בָּכָה, בָּכְתָה (бахíті, бахá, бахтá)
- майбутній час: אֶבְכֶּה, יִבְכֶּה, נִבְכֶּה (евкé, ївкé, нівкé)

## Короткі приклади

- הַיֶּלֶד בּוֹכֶה. — Дитина плаче.
- הִיא בָּכְתָה כָּל הָעֶרֶב. — Вона плакала весь вечір.
- אִם זֶה יִכְאַב, נִבְכֶּה קְצָת. — Якщо болітиме, ми трохи поплачемо.
