# У басейні

בַּקַּיִץ נוֹעָה הוֹלֶכֶת לַבְּרֵכָה עִם חֲבֵרָה.
הֵן מֵבִיאוֹת מַגֶּבֶת, מַיִם וּמִשְׁקְפֵי שְׁחִיָּה.
לִפְנֵי הַכְּנִיסָה הֵן יוֹשְׁבוֹת קְצָת בַּצֵּל.

אַחַר כָּךְ הֵן נִכְנָסוֹת לַמַּיִם וְשׂוֹחוֹת לְאַט.
נוֹעָה עוֹצֶרֶת בַּקָּצֶה וְצוֹחֶקֶת כְּשֶׁחֲבֵרָהּ מַתִּיזָה מַיִם.
זֶה אַחַד הַיָּמִים הַכִּיפִּים שֶׁל הַקַּיִץ.

## Основні слова

- בְּרֵכָה - басейн
- מַגֶּבֶת - рушник
- מִשְׁקְפֵי שְׁחִיָּה - окуляри для плавання
- צֵל - тінь
- קָצֶה - край
- קַיִץ - літо

- לִשְׂחוֹת - плавати
- לְהַתִּיז - бризкати
