# Запізнення на автобус

הַבֹּקֶר רָן יוֹצֵא מַהַבַּיִת מְאֻחָר.
הוּא הוֹלֵךְ מַהֵר לַתַּחֲנָה עִם הַתִּיק עַל הַגַּב.
כְּשֶׁהוּא מַגִּיעַ, הָאוֹטוֹבּוּס כְּבָר נוֹסֵעַ.

רָן נוֹשֵׁם עָמֹק וּמַבִּיט עַל הַשָּׁעוֹן.
הוּא מַחְלִיט לְחַכּוֹת לָאֹטוֹבּוּס הַבָּא.
בִּזְמַן הַהַמְתָּנָה הוּא קוֹרֵא הוֹדָעָה בַּטֵּלֵפוֹן.

## Основні слова

- תַּחֲנָה - зупинка
- תִּיק - сумка
- גַּב - спина
- אוֹטוֹבּוּס - автобус
- שָׁעוֹן - годинник
- הַמְתָּנָה - очікування

## Базові дієслова

- לָצֵאת - виходити
- לָלֶכֶת - йти
- לְהַגִּיעַ - прибувати
- לִנְשֹׁם - дихати
- לְהַחְלִיט - вирішувати
- לְחַכּוֹת - чекати
