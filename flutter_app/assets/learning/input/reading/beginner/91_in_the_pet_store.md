# У зоомагазині

הַיּוֹם נוֹעָה נִכְנֶסֶת לַחֲנוּת חַיּוֹת.
הִיא רוֹצָה לִקְנוֹת אֹכֶל לַדָּגִים שֶׁלָּהּ.
בַּחֲנוּת יֵשׁ גַּם צִפּוֹרִים, אַרְנָב וְחֲתוּלִים קְטַנִּים.

נוֹעָה עוֹמֶדֶת קְצָת לְיַד הָאַקְוַרְיוּם וּמַבִּיטָה בַּדָּגִים.
הַמּוֹכֵר מַרְאֶה לָהּ אֵיזֶה אֹכֶל מַתְאִים לָהֶם.
הִיא מְשַׁלֶּמֶת וְחוֹזֶרֶת הַבַּיְתָה.

## Основні слова

- חֲנוּת חַיּוֹת - зоомагазин
- דָּג / דָּגִים - риба / риби
- צִפּוֹר - птах
- אַרְנָב - кролик
- חָתוּל - кіт
- אַקְוַרְיוּם - акваріум
