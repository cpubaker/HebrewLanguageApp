# Чекаю друга

לֵאָה מַגִּיעָה לְבֵית קָפֶה חָמֵשׁ דַּקּוֹת מֻקְדָּם.
הִיא קוֹבַעַת שָׁם לְפַגֵּשׁ חֲבֵרָה יְשָׁנָה.
עַד שֶׁהַחֲבֵרָה בָּאָה, לֵאָה יוֹשֶׁבֶת לְיַד הַחַלּוֹן.

הִיא מַבִּיטָה בָּאֲנָשִׁים שֶׁעוֹבְרִים בָּרְחוֹב.
אַחַר כַּמָּה דַּקּוֹת הַחֲבֵרָה נִכְנֶסֶת וּמְחַבֶּקֶת אוֹתָהּ.
שְׁתֵּיהֶן שְׂמֵחוֹת מְאֹד לִפָּגֵשׁ.

## Основні слова

- בֵּית קָפֶה - кафе
- חָבֵר / חֲבֵרָה - друг / подруга
- חַלּוֹן - вікно
- אֲנָשִׁים - люди
- רְחוֹב - вулиця
- חִבּוּק - обійми
