# Перша книжка івритом

הַיּוֹם לֵאָה פּוֹתַחַת סֵפֶר רִאשׁוֹן בְּעִבְרִית.
הַסֵּפֶר לֹא גָּדוֹל, וְיֵשׁ בּוֹ הַרְבֵּה תְּמוּנוֹת.
בַּהַתְחָלָה הִיא קוֹרֵאת לְאַט וּבוֹדֶקֶת מִלִּים חֲדָשׁוֹת.

אַחַר כַּמָּה עַמּוּדִים הִיא מַרְגִּישָׁה יוֹתֵר בִּטָּחוֹן.
הִיא מְחַיֶּכֶת, כִּי הִיא מֵבִינָה יוֹתֵר מִמָּה שֶׁחָשְׁבָה.
זֶה צַעַד קָטָן וְחָשׁוּב בַּלִּמּוּד שֶׁלָּהּ.

## Основні слова

- סֵפֶר רִאשׁוֹן - перша книжка
- עִבְרִית - іврит
- תְּמוּנָה - малюнок
- מִלָּה חֲדָשָׁה - нове слово
- בִּטָּחוֹן - впевненість
- צַעַד - крок

## Базові дієслова

- לִפְתּוֹחַ - відкривати
- לִקְרוֹא - читати
- לִבְדּוֹק - перевіряти
- לְהַרְגִּישׁ - відчувати
- לְהָבִין - розуміти
- לַחְשֹׁב - думати
