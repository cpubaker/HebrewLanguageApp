# В аптеці

לֵאָה הוֹלֶכֶת לְבֵית הַמִּרְקַחַת.
הִיא לֹא מַרְגִּישָׁה כָּל כָּךְ טוֹב הַיּוֹם.
כּוֹאֵב לָהּ הָרֹאשׁ, וְהִיא רוֹצָה עֶזְרָה.
הָרוֹקַחַת אוֹמֶרֶת לָהּ שָׁלוֹם וְשׁוֹאֶלֶת מַה כּוֹאֵב.

לֵאָה אוֹמֶרֶת, "כּוֹאֵב לִי הָרֹאשׁ."
הָרוֹקַחַת נוֹתֶנֶת לָהּ תְּרוּפָה וְאוֹמֶרֶת לִשְׁתּוֹת מַיִם.
לֵאָה לוֹקַחַת אֶת הַתְּרוּפָה, אוֹמֶרֶת תּוֹדָה וְחוֹזֶרֶת הַבַּיְתָה.

- עֶזְרָה - допомога
- רוֹקַחַת - фармацевтка
- תְּרוּפָה - ліки

- יוֹם - день

- טוֹב - добре
