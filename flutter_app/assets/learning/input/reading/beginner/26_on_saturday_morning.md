# Суботній ранок

בְּשַׁבָּת בַּבֹּקֶר הַבַּיִת שָׁקֵט.
אַבָּא יוֹשֵׁב בַּמִּרְפֶּסֶת וְשׁוֹתֶה קָפֶה.
אִמָּא קוֹרֵאת עִתּוֹן וְהַשֶּׁמֶשׁ נִכְנֶסֶת מִן הַחַלּוֹן.
אֲנִי קָם לְאַט וְהוֹלֵךְ לַמִּטְבָּח.

עַל הַשֻּׁלְחָן יֵשׁ לֶחֶם, גְּבִינָה וְתֵּה חַם.
אַחַר הָאֹכֶל אֲנִי יוֹשֵׁב עִם הַמִּשְׁפָּחָה וּמְדַבֵּר קְצָת.
טוֹב לִהְיוֹת בַּבַּיִת בְּבֹקֶר שֶׁל שַׁבָּת.
