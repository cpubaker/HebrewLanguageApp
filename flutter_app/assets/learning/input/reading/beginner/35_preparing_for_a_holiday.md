# Готуємося до свята

הַיּוֹם הַמִּשְׁפָּחָה מְכִינָה אֶת הַבַּיִת לַחַג.
אִמָּא מְנַקָּה אֶת הַשֻּׁלְחָן, וְאַבָּא מְסַדֵּר אֶת הַכִּסְאוֹת.
עַל הַשֻּׁלְחָן יֵשׁ מַפָּה לְבָנָה, פְּרָחִים וְנֵרוֹת.
אֲנִי וְאָחוֹתִי מְבִיאִים לֶחֶם וְגַם עֻגָּה.

אַחַר כָּךְ כֻּלָּם מַבִּיטִים בַּחֶדֶר וְאוֹמְרִים, "כַּמָּה יָפֶה!"
הַבַּיִת נָקִי, מוּכָן וּמָלֵא אוֹר.
טוֹב לְהָכִין יַחַד אֶת הַבַּיִת לַחַג.

## Основні слова

- חַג - свято
- מִשְׁפָּחָה - сім'я
- מַפָּה - скатертина
- לָבָן / לְבָנָה - білий / біла
- פֶּרַח / פְּרָחִים - квітка / квіти
- נֵר / נֵרוֹת - свічка / свічки
- חֶדֶר - кімната
- נָקִי - чистий
- מוּכָן - готовий
- אוֹר - світло

## Базові дієслова

- לְהָכִין - готувати
- לְנַקּוֹת - прибирати, чистити
- לְסַדֵּר - розставляти, упорядковувати
- לְהָבִיא - приносити
- לְהַבִּיט - дивитися
- לוֹמַר - казати
