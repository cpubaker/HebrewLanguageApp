# Недільне прання

בְּיוֹם רִאשׁוֹן תָּמָר עוֹשָׂה כְּבִיסָה.
הִיא מְבִיאָה אֶת הַבְּגָדִים לַמְּכוֹנָה.
יֵשׁ שָׁם חוּלְצוֹת, גַּרְבַּיִם וּמְגַבּוֹת קְטַנּוֹת.

תָּמָר שָׂמָה סַבּוֹן וּבוֹחֶרֶת תָּכְנִית קְצָרָה.
אַחַר כָּךְ הִיא תּוֹלָה אֶת הַבְּגָדִים לְיַבֵּשׁ.
בָּעֶרֶב הַכֹּל נָקִי וְרֵיחַ טוֹב בַּבַּיִת.

## Основні слова

- יוֹם רִאשׁוֹן - неділя
- כְּבִיסָה - прання
- בְּגָדִים - одяг
- מְכוֹנָה - машина
- סַבּוֹן - мило
- רֵיחַ - запах

## Базові дієслова

- לַעֲשׂוֹת - робити

- לִבְחוֹר - вибирати
- לִתְלוֹת - вішати
- לְיַבֵּשׁ - сушити
