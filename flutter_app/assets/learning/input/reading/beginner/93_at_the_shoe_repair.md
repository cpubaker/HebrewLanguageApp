# Ремонт взуття

לְרוֹנִי יֵשׁ נַעַל אַחַת קְרוּעָה.
הוּא הוֹלֵךְ לַחֲנוּת תִּקּוּן נַעֲלַיִם בָּרְחוֹב הָרָאשִׁי.
הַסַּנְדְלָר מַבִּיט בַּנַּעַל וְאוֹמֵר שֶׁאֶפְשָׁר לְתַקֵּן אוֹתָהּ.

רוֹנִי שׁוֹאֵל מָתַי הַנַּעַל תִּהְיֶה מוּכָנָה.
הַסַּנְדְלָר אוֹמֵר, "מָחָר בַּצָּהֳרַיִם."
רוֹנִי מוֹדֶה לוֹ וְחוֹזֵר הַבַּיְתָה.

## Основні слова

- נַעַל - черевик
- קָרוּעַ / קְרוּעָה - порваний / порвана
- תִּקּוּן - ремонт
- סַנְדְלָר - швець
- רְחוֹב רָאשִׁי - головна вулиця
- צָהֳרַיִם - полудень
