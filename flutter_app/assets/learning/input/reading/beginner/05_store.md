# Магазин

דָּנָה הוֹלֶכֶת לַחֲנוּת.
הִיא קוֹנָה פֵּרוֹת.
הִיא לוֹקַחַת תַּפּוּחַ אָדֹם.
הִיא לוֹקַחַת בָּנָנָה צְהֻבָּה.
הִיא לוֹקַחַת תַפּוּז כָּתֹם.
הַקּוֹפָאִי נוֹתֵן לָהּ שַׂקִּית.
דָּנָה מְשַׁלֶּמֶת וְחוֹזֶרֶת הַבַּיְתָה.
