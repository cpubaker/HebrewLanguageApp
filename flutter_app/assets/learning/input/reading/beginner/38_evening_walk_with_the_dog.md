# Вечірня прогулянка з собакою

נוֹעָה יוֹצֵאת בָּעֶרֶב עִם הַכֶּלֶב שֶׁלָּהּ.
הָרְחוֹב שָׁקֵט, וְהָאֲוִיר קָרִיר וְנָעִים.
הַכֶּלֶב הוֹלֵךְ לְיַד נוֹעָה וּמְרִיחַ אֶת הַדֶּשֶׁא.

נוֹעָה עוֹצֶרֶת לְיַד הַגִּנָּה וּמַבִּיטָה בַּפְּרָחִים.
אַחַר כָּךְ הִיא חוֹזֶרֶת הַבַּיְתָה בְּשָׂמְחָה.
זֹאת הֲלִיכָה קְטַנָּה אֲבָל טוֹבָה מְאֹד.

## Основні слова

- עֶרֶב - вечір
- כֶּלֶב - собака
- רְחוֹב - вулиця
- אֲוִיר - повітря
- גִּנָּה - сад
- פֶּרַח / פְּרָחִים - квітка / квіти
