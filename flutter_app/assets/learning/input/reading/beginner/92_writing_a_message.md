# Пишемо повідомлення

לֵאָה רוֹצָה לִכְתּוֹב הוֹדָעָה לַמּוֹרָה.
הִיא פּוֹתַחַת אֶת הַטֵּלֵפוֹן וּמַתְחִילָה לִכְתּוֹב לְאַט.
הַהוֹדָעָה קְצָרָה: הִיא מַסְבִּירָה שֶׁמָּחָר הִיא תַּגִּיעַ מְאֻחָר.

לִפְנֵי הַשְּׁלִיחָה הִיא קוֹרֵאת אֶת הַטֶּקְסְט עוֹד פַּעַם.
אַחַר כָּךְ הִיא לוֹחֶצֶת עַל שְׁלַח.
כְּבָר עוֹד רֶגַע אַחַר כָּךְ מַגִּיעָה תְּשׁוּבָה קְצָרָה.

## Основні слова

- הוֹדָעָה - повідомлення
- מוֹרָה - вчителька
- טֶּלֵפוֹן - телефон
- טֶּקְסְט - текст
- שְׁלַח - надіслати
- תְּשׁוּבָה - відповідь

## Базові дієслова

- לִרְצוֹת - хотіти
- לִכְתּוֹב - писати
- לִפְתּוֹחַ - відкривати
- לְהַסְבִּיר - пояснювати
- לִקְרוֹא - читати
- לִלְחוֹץ - натискати
