# Одяг

לֵאָה פּוֹתַחַת אֶת הָאָרוֹן בַּבֹּקֶר.
הִיא רוֹאָה שָׁם חוּלְצָה, מִכְנָסַיִם, שַׂמְלָה וְמִיל.
הַיּוֹם קְצָת קַר, וְלָכֵן הִיא לוֹבֶשֶׁת מִכְנָסַיִם וּמִיל.

לִפְעָמִים הִיא לוֹבֶשֶׁת חוּלְצָה לְבָנָה אוֹ שַׂמְלָה כְּחוּלָּה.
יֵשׁ לָהּ גַּם נַעֲלַיִם שְׁחוֹרוֹת וְכּוֹבַע קָטָן.
בַּחוּרֶף הִיא שָׂמָה גַּם גַּרְבַּיִם חַמּוֹת.

אָחִיהָ לוֹבֵשׁ חוּלְצָה, מִכְנָסַיִם וְנַעֲלַיִם.
לִפְנֵי שֶׁהוּא יוֹצֵא מֵהַבַּיִת, הוּא מַבִּיט בַּמַּרְאָה.
הוּא רוֹצֶה לִהְיוֹת נָקִי וּמְסֻדָּר.

לִלְבּוֹשׁ בְּגָדִים מַתְאִימִים זֶה חָשׁוּב בְּכָל יוֹם.

## Основні слова

- אָרוֹן - шафа
- חוּלְצָה - сорочка, футболка, блузка
- מִכְנָסַיִם - штани
- שַׂמְלָה - сукня
- מִיל - пальто
- נַעֲלַיִם - взуття, туфлі
- כּוֹבַע - капелюх, шапка
- גַּרְבַּיִם - шкарпетки
- בֶּגֶד / בְּגָדִים - одяг / речі, одяг
- מַרְאָה - дзеркало
- לָבָן / לְבָנָה - білий / біла
- שָׁחוֹר / שְׁחוֹרוֹת - чорний / чорні
- כָּחוֹל / כְּחוּלָּה - синій / синя
- קַר - холодно, холодний
- חַם / חַמּוֹת - теплий / теплі
