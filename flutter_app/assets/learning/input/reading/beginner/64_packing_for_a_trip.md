# Складаємо валізу

מָחָר הַמִּשְׁפָּחָה יוֹצֵאת לְנְסִיעָה קְצָרָה.
בָּעֶרֶב תָּמָר פּוֹתַחַת אֶת הַמַּזְוָדָה עַל הַמִּטָּה.
הִיא שָׂמָה בְּגָדִים, מִבְרֶשֶׁת שִׁנַּיִם וְסֵפֶר קָטָן.

אִמָּא מַזְכִּירָה לָהּ גַּם לָקַחַת כוֹבַע וּמַיִם.
תָּמָר בּוֹדֶקֶת שֶׁהַכֹּל בִּפְנִים וְסוֹגֶרֶת אֶת הַמַּזְוָדָה.
עַכְשָׁו הִיא מוּכָנָה לְמָחָר.

## Основні слова

- מַזְוָדָה - валіза
- נְסִיעָה - поїздка
- בְּגָדִים - одяг
- מִבְרֶשֶׁת שִׁנַּיִם - зубна щітка
- כּוֹבַע - капелюх
- מוּכָן / מוּכָנָה - готовий / готова

## Базові дієслова

- לָצֵאת - вирушати
- לִפְתּוֹחַ - відкривати
- לָשִׂים - класти
- לְהַזְכִּיר - нагадувати
- לִבְדּוֹק - перевіряти
- לִסְגּוֹר - закривати
