# Маленьке свято вдома

הַיּוֹם יֵשׁ לְדָּנִי יוֹם הֻלֶּדֶת.
בָּעֶרֶב הַמִּשְׁפָּחָה עוֹשָׂה חֲגִיגָה קְטַנָּה בַּבַּיִת.
עַל הַשֻּׁלְחָן יֵשׁ עוּגָה, נֵרוֹת וְגַם בַּלוֹנִים.

חֲבֵרִים אֶחָדִים בָּאִים וְאוֹמְרִים, "מַזָּל טוֹב!"
דָּנִי מְבַקֵּשׁ מַשָּׁאָלָה וּמְכַבֶּה אֶת הַנֵּרוֹת.
כֻּלָּם אוֹכְלִים, שָׁרִים וּשְׂמֵחִים יַחַד.

## Основні слова

- יוֹם הֻלֶּדֶת - день народження
- חֲגִיגָה - святкування
- נֵר / נֵרוֹת - свічка / свічки
- בַּלוֹן - кулька
- מַזָּל טוֹב - вітаю
- מַשָּׁאָלָה - бажання

## Базові дієслова

- לַעֲשׂוֹת - робити
- לָבוֹא - приходити
- לוֹמַר - казати
- לְבַקֵּשׁ - загадувати, просити
- לְכַבּוֹת - гасити
- לָשִׁיר - співати
