# Ранок у ванній

בַּבֹּקֶר דָּנִי הוֹלֵךְ לַחֲדַר הָרַחְצָה.
הוּא פּוֹתֵחַ אֶת הַבַּרְז וְרוֹאֶה מַיִם חַמִּים.
הוּא רוֹחֵץ פָּנִים וְיָדַיִם.
אַחַר כָּךְ הוּא לוֹקֵחַ מַגֶּבֶת רַכָּה.

דָּנִי מִסְתַּכֵּל בַּמַּרְאָה וּמְחַיֵּךְ.
עַל הַכִּיּוֹר יֵשׁ סַבּוֹן, מִבְרֶשֶׁת שִׁנַּיִם וְכוֹס קְטַנָּה.
בַּסּוֹף הוּא סוֹגֵר אֶת הַמַּיִם וְיוֹצֵא מִן הַחֶדֶר.

- יָדַיִם - руки
- מַגֶּבֶת - рушник
- רַךְ / רַכָּה - м'який / м'яка
- מַרְאָה - дзеркало
- כִּיּוֹר - умивальник
- סַבּוֹן - мило
- מִבְרֶשֶׁת שִׁנַּיִם - зубна щітка
