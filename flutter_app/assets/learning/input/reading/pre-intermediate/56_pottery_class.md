# Урок кераміки

יעל מגיעה היום לשיעור קדרות כדי להשתתף במפגש וללמוד משהו חדש.
על השולחנות יש גלגל אובניים, חתיכות חימר ומדף ייבוש, וכל המשתתפים מתכוננים להתחלה.
המדריך פותח בהסבר קצר, ויעל מנסה להבין איך לחבר בין החומר העיוני לבין התרגול.
בהמשך היא עובדת עם שותף, שואלת כמה שאלות ומתקנת פרטים קטנים תוך כדי תנועה.
לקראת סוף השיעור היא כבר מרגישה בטוחה יותר ורואה שהקצב מתאים לה.
כשהמפגש נגמר, היא רושמת לעצמה מה כדאי לתרגל גם בבית.

## Основні слова

- קדרות - кераміка
- גלגל אובניים - гончарне коло
- חימר - глина
- מדף ייבוש - полиця для сушіння
- קערה קטנה - маленька миска
- ידיים רטובות - мокрі руки
- צורה ראשונית - початкова форма
- עבודה עדינה - делікатна робота

## Нові дієслова

- להשתתף - брати участь
- להקשיב - слухати
- לתרגל - практикувати
- לתקן - виправляти
- לשפר - покращувати
