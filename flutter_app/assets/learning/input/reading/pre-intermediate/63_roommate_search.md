# Пошук сусіда по квартирі

יעל מגיעה היום למפגש היכרות עם שותפים פוטנציאליים כדי לטפל בעניין חשוב שלא רצתה לדחות.
במקום יש שולחן משותף, רשימת שאלות ומודעת דירה, ולכן צריך לשים לב לכל פרט קטן.
בהתחלה היא בודקת מה בדיוק צריך לעשות, ואז מדברת עם הנציג הראשון שפנוי לעזור לה.
אחר כך היא עוברת שלב אחרי שלב, ממתינה כמה דקות ומוודאת שכל המסמכים מסודרים.
באמצע התהליך מתברר שיש פרט אחד שחסר, אבל בעזרת הסבר ברור היא מצליחה להשלים אותו.
בסוף היא מקבלת תשובה מסודרת ויוצאת בתחושה שהזמן נוצל היטב.

## Основні слова

- שותף לדירה - сусід по квартирі
- מפגש היכרות - знайомча зустріч
- רשימת שאלות - список запитань
- מודעת דירה - оголошення про квартиру
- חלוקת הוצאות - поділ витрат
- לוח זמנים אישי - особистий графік
- הרגלי מגורים - побутові звички
- החלטה משותפת - спільне рішення

## Нові дієслова

- להמתין - чекати
- לבדוק - перевіряти
- למסור - подавати, передавати
- להשלים - доповнювати, завершувати
- לקבל - отримувати
